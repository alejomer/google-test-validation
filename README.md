# Technical challenge - Google search validations

## Installation

Install all dependencies with npm

```bash
  npm install
```
 
## Commands for execution

run on Cypress Test Runner 

```bash
  npm run cy:open
```

run in  Terminal

```bash
  npm run cy:run
```
